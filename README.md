# Django User Authentication API

一个基于 Django REST framework 和 JWT 的用户认证系统。

## 功能特点

- 用户注册
- 用户登录（JWT认证）
- 获取用户信息

## 快速开始

### 方式一：直接运行

### 方式二：使用 Docker
1. 克隆项目并进入项目目录
```bash
git clone 
cd runwu_demo
 ```

2. 使用 Docker Compose 启动服务
```bash
docker compose up --build
 ```
